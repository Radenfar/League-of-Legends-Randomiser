League of Legends Randomiser
-------------------------------
No dependencies! Just unzip and run app.py.

This is a fun programming that gives completely randomised champions, roles, spells, items, ability max order, runes and even shards.
Create fun challenges in games and discover completely hidden great builds!

Simple run app.py and it will print out your game options!
The program ends with an input so that you can read it as much as you want, simply hit enter to close the program.
